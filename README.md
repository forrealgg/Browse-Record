# Browse Recorder - An all in one Screen Recorder Fully made in browser.
## About
Browse-Recorder is an all in one Screen Recorder Fully made in browser! Supporting Window only Recording or Full Screen Recording.
## Features

- Dark styled UI
- Fullscreen mode
- Window only mode
- Cross platform


## How to use?
1. Open [the app](https://forrealgg.github.io/Browse-Record/) in your browser.
2. Click Start Recording to begin capturing your screen.
3. Choose what you want to share (entire screen, window, or tab).
4. Click Stop Recording when finished.
5. Your recording will automatically appear below.
